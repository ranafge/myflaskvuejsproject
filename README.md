# 🚀 Flask + Vue.js + MySQL Full-Stack Application

A complete full-stack web application for article management with **Flask REST API** backend, **Vue.js 3** frontend, and **MySQL** database. Fully containerized with Docker for easy deployment.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📝 **Full CRUD** | Create, Read, Update, Delete articles |
| 🐍 **Flask Backend** | RESTful API with SQLAlchemy ORM |
| ⚡ **Vue.js Frontend** | Modern SPA with Vue 3 Composition API |
| 🗄️ **MySQL Database** | Persistent data storage |
| 🐳 **Docker Ready** | Complete containerization |
| 🔄 **CORS Enabled** | Secure cross-origin requests |
| 🇧🇩 **Bangladesh Timezone** | Pre-configured for Asia/Dhaka |
| 📱 **Responsive Design** | Works on all devices |
